# Greet App

A simple full-stack greeting app using **Node.js, Express, and React**.  
The backend serves a greeting API, and the frontend provides a user interface to interact with it.

- Backend built with **Node.js & Express**.
- Frontend using **React & CSS** (No Tailwind CSS).
- Users can enter their name and get a personalized greeting.
- Displays errors if the name is missing or if there's a server issue.


## backend Setup
cd greet-app-backend
npm install
node server.js

## frontend Setup
cd greet-app-frontend
npm install
npm start


## how to run the code
Start both the backend and frontend.
Enter your name in the frontend input field.
Click "Get Greeting".
The response from the API will be displayed.


## Tech Stack
Backend: Node.js, Express
Frontend: React, CSS
Tools: Fetch API, JSX, useState
